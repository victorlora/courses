Welcome to the help page.
This website has been designed to allow for an easier method to creating a professional resume.
To create a resume, simply fill out the form sheet with all of your personal information.
Make sure all inputs contain letters, with no numbers added.
If any text box contains wrong input information, an error message will display at the top, describing what is incorrect.
From there press "submit" and your form you should formatted as a full resume containing all of the information filled in from the form.
